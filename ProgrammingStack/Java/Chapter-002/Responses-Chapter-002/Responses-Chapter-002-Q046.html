Code blocks in Java are created by enclosing two or more java statements within opening and closing curly braces ({ }).

Code blocks can be used at any place where a single java statement can be used.

Example,

The syntax of for loop:-

for(initialization; condition; iteration) statement;

can be replaced as:-

for(initialization; condition; iteration)
{
    statement-1;
    statement-2;
    statement-3;
}
