The 8 different types of data supported by Java can be divided into 4 different categories:-

1. The category representing whole numbers(both +ve and -ve) called Integers. The data types byte, short, int and long fall under this category.

2. The category representing fractional numbers(both +ve and -ve) called Floating Point Numbers. The data types float and double fall under this category.

3. The category representing symbols called Characters. The data type char falls under this category.

4. The category representing binary values (true/false) called Boolean. The data type boolean falls under this category.
