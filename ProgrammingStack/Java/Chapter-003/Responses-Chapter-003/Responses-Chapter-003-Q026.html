1. There is no way of representing a byte literal. Nothing like byte literal exists.

2. There is no way of representing a short literal. Nothing like short literal exists.

3. Any integer constant by default is a integer literal. Example,
100, 45676 are integer literals.

4. A long literal is represented by appending capital 'el' or small 'el' to the integer literal.Example,
100l, 100L are long literals

Note: There is no way of converting an integer literal to byte literal or short literal.
