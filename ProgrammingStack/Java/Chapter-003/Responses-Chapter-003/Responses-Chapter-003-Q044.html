The characters represented using escape sequences are expected to be used extensively. Since it would be difficult for java users to remember decimal/octal/hexadecimal values of each of these characters, therefore escape sequences were created.

For example,

NEWLINE character can be represented as.

char ch = 10;
char ch = '\012';
char ch = '\u000A';

Using escape sequence it can be represented as,

char ch = '\n';
