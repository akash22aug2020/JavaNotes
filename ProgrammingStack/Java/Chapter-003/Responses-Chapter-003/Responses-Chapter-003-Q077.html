Regular array declaration syntax: type array-name[];

Alternate array declaration syntax: type[] array-name;

This alternate array declaration syntax is particularly useful in the following cases:-

1. When declaring multiple arrays of the same type together: Eg, int[] a,b,c declares 3 arrays. Using the regular way, this would have been written as int a[], b[], c[]

2. When declaring return type of methods: Eg, public static int[] sortArray()
