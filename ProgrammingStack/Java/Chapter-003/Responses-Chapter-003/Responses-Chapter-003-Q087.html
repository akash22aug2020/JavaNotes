Following are the restrictions on using 'var' for automatic type inference of local variables:-

1. The initializer cannot be null. Example, var i = null; is not allowed.

2. An array cannot be initialized when declared using the 'var' keyword. Example, var a = {1, 2, 3}; is not allowed.

3. 'var' cannot be used as the name of the class, interface, enumeration and annotation.

4. 'var' cannot be used to declare the type of the exception caught by the catch statement. Example, catch(var Exception){} is not allowed.

5. Lambda expressions or method references cannot be used as initializers.
